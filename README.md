# Design Skills for Claude Code

A curated collection of design-focused skills organized by category.

---

## Core Design System
| Skill | Description |
|-------|-------------|
| `soft-visual-design` | Establishes fonts, spacing, shadows, card structures, and animations |
| `taste-frontend` | Senior UI/UX engineer rules, strict component architecture |
| `frontend-design` | Production-grade frontend interfaces with high design quality |
| `tailwind-design-system` | Design tokens, component libraries, responsive patterns |

## Creative Process
| Skill | Description |
|-------|-------------|
| `brainstorming` | Explore ideas before any creative or architectural work |
| `simple` | Streamlined brainstorming for quick decisions |
| `critique` | UX evaluation — hierarchy, cognitive load, emotional resonance |

## Build & Layout
| Skill | Description |
|-------|-------------|
| `landing-page` | Complete, deployment-ready landing pages |
| `redesign` | Upgrades existing sites to premium quality |
| `arrange` | Fix layout, spacing, and visual rhythm |
| `adapt` | Responsive design across screen sizes and devices |

## Refinement & Polish
| Skill | Description |
|-------|-------------|
| `polish` | Final quality pass — alignment, spacing, micro-details |
| `distill` | Strip to essence, remove unnecessary complexity |
| `simplify` | Review for reuse, quality, and efficiency |
| `normalize` | Realign UI to design system standards |
| `quieter` | Tone down overstimulating designs |
| `clarify` | Improve UX copy, labels, and error messages |

## Visual Enhancement
| Skill | Description |
|-------|-------------|
| `colorize` | Add strategic color to monochromatic interfaces |
| `bolder` | Amplify safe or boring designs for more impact |
| `delight` | Add joy, personality, and unexpected touches |
| `overdrive` | Technically ambitious — shaders, physics, scroll-driven reveals |

## Motion & Animation
| Skill | Description |
|-------|-------------|
| `ux-motion` | Motion design and animation implementation |
| `animate` | Enhance features with purposeful animations |

## Brand & Identity
| Skill | Description |
|-------|-------------|
| `brand-guide-extractor` | Scrape brand data from any website |
| `logo-creator` | AI-generated logos with iteration |
| `banner-creator` | AI-generated banners |

## Specialized UI Styles
| Skill | Description |
|-------|-------------|
| `brutalist-ui` | Raw mechanical Swiss/military aesthetic |
| `minimalist-ui` | Clean editorial, warm monochrome, bento grids |

## Quality Assurance
| Skill | Description |
|-------|-------------|
| `audit` | Accessibility, performance, theming, responsive checks |
| `web-design-guidelines` | Review UI for Web Interface Guidelines compliance |
| `dogfood` | Systematically explore and test for bugs and UX issues |

---

## Recommended Pipeline

```
brainstorming / simple
    → soft-visual-design / taste-frontend
        → frontend-design
            → polish / distill
```
